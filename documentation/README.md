# ML CLI Documentation Site

Dynamic documentation is generated from `ml.bat` using the parser pipeline.

## Build

```bash
npm run docs:build
```

## Watch

```bash
npm run docs:watch
```

## Open

Open `documentation/index.html` in your browser.

## Data Source

Generated data file:

- `documentation/assets/data/commands.json`

Parser script:

- `scripts/parse-ml.js`
